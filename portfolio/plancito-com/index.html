Client

Plancito.com is a social event planning app designed to simplify group gatherings. It allows users to create events, manage invites, collaborate on playlists, split expenses, and organize tasks seamlessly.

The challenge

Building a collaborative, real-time platform that handles multiple event management features while keeping the UX intuitive and mobile-friendly.

My solution

Implemented a full-stack solution with React, Supabase, and TailwindCSS, integrating real-time updates, expense tracking, and task management features into a cohesive user experience.
